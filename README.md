## Installation

```shell
npm install
```

## Parameters

|          | default |
| -------- | :-----: |
| job      |  React  |
| location | Berlin  |

## Env variables for LinkedInScraper

create .env file in root of project

| name     |          |    description    |
| -------- | :------: | :---------------: |
| login    | required |  linkedIn login   |
| password | required | linkedIn password |
| path     | optional |  path to browser  |

## Usage

```shell
npm start
```

or with parameters

```shell
npm start React London
```
